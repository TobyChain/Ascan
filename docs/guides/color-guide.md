## 🎯 使用场景

### 标题
```css
h1 { color: #1a1a1a; }  /* 页面主标题 */
h2 { color: #1a1a1a; }  /* 章节标题 */
h3 { color: #303133; }  /* 小节标题 */
h4 { color: #303133; }  /* 子标题 */
```

### 正文
```css
p, div { color: #303133; }  /* 常规文本 */
label { color: #303133; }   /* 表单标签 */
```

### 交互元素
```css
button { 
  background: #409eff; 
  color: #ffffff; 
}
button:hover { 
  background: #66b1ff; 
}
```

### 卡片内容
```css
.card {
  background: #ffffff;
  color: #303133;
}
```

## 🔧 自定义配色

如果需要调整配色，修改 `web/app.py` 中的CSS变量：

```python
st.markdown("""
<style>
    /* 修改主色调 */
    --primary-color: #409eff;
    
    /* 修改文字颜色 */
    --text-primary: #1a1a1a;
    --text-regular: #303133;
    --text-secondary: #606266;
    
    /* 修改背景色 */
    --bg-page: #f5f7fa;
    --bg-card: #ffffff;
</style>
""", unsafe_allow_html=True)
```
