## 🎨 组件配色详情

### 1. 标题系统
```css
h1 {
  color: #ffffff;           /* 纯白，最高对比度 */
  border-bottom: #1f77b4;   /* 蓝色下划线 */
}

h2 {
  color: #f0f0f0;           /* 亮白 */
}

h3, h4 {
  color: #e8eaed;           /* 浅灰白 */
}
```

### 2. 卡片组件
```css
.card {
  background: #2d3139;      /* 深灰背景 */
  border: 1px solid #3a3f4b; /* 深灰边框 */
  color: #e8eaed;           /* 浅灰白文字 */
}
```

### 3. 按钮
```css
.button {
  background: #1f77b4;      /* 蓝色背景 */
  color: #ffffff;           /* 白色文字 */
}

.button:hover {
  background: #2e8bc0;      /* 亮蓝背景 */
  box-shadow: 0 4px 12px rgba(31, 119, 180, 0.4);
}
```

### 4. 输入框
```css
input {
  background: #2d3139;      /* 深灰背景 */
  color: #ffffff;           /* 白色文字 */
  border: 2px solid #3a3f4b; /* 深灰边框 */
}

input:focus {
  border-color: #1f77b4;    /* 蓝色边框 */
  box-shadow: 0 0 0 2px rgba(31, 119, 180, 0.2);
}

input::placeholder {
  color: #6c757d;           /* 深灰占位符 */
}
```

### 5. 推荐标签
```css
.tag-high {
  background: #ff6b6b;      /* 红色 - 极度推荐 */
  color: #ffffff;
}

.tag-medium {
  background: #e6a23c;      /* 橙色 - 很推荐 */
  color: #ffffff;
}

.tag-low {
  background: #67c23a;      /* 绿色 - 推荐 */
  color: #ffffff;
}
```
