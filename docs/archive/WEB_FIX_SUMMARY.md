### 1. "机构"信息显示问题
**问题**: Web界面显示"机构信息暂缺"

**原因**: 
- Web界面的 `render_paper_card` 函数中有显示机构信息的代码
- 机构信息可能为空，显示"机构信息暂缺"

**解决方案**: 
- 移除了论文卡片中的机构信息显示
- 移除了"查看详情"中的机构信息显示

### 2. HTML渲染问题
**问题**: Web界面可能显示HTML代码

**原因**: 
- 没有对HTML内容进行转义
- 可能存在XSS攻击风险

**解决方案**: 
- 使用 `html.escape()` 对所有用户输入的内容进行转义
- 确保所有HTML内容都经过转义
