## 🎯 问题分析

### 1. HTML渲染问题
**问题**: Web界面显示HTML代码
```
</p> <p style="color: #888; font-size: 0.8rem; margin: 0;"> 📅 2026-02-03 | 🏷️ rag, agent </p> </div> <div style="text-align: right; margin-left: 1rem;"> <span class="recommendation-high">极度推荐</span> <br/> <small style="color: #888;">rag</small> </div> </div>
```

**原因**: 
- 使用了`st.markdown()`渲染HTML
- 使用了`unsafe_allow_html=True`
- HTML标签被直接显示

**解决方案**: 
- 移除了HTML渲染
- 使用Streamlit原生组件代替
- 使用`st.markdown()`渲染纯文本

### 2. "机构"信息显示问题
**问题**: Web界面显示"机构信息暂缺"

**原因**: 
- Web界面的 `render_paper_card` 函数中有显示机构信息的代码
- 机构信息可能为空，显示"机构信息暂缺"

**解决方案**: 
- 移除了论文卡片中的机构信息显示
- 移除了"查看详情"中的机构信息显示
